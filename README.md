# Sales Data Analysis Using Python

I created this project as part of my data analysis practice to understand how real-world sales data works.
Through this project, I learned how to clean messy data and convert it into meaningful insights.

## 🎯 Objective

The main objective of this project is to analyze sales data and answer important business questions like:

* Which products sell the most?
* Which payment method is used more?
* Who are repeat customers?
* How sales change over time?

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 🔧 Steps Performed

### 1️⃣ Data Cleaning

* Removed missing and incorrect values
* Fixed data types (dates, numbers)
* Handled duplicate records
* Standardized text fields

### 2️⃣ Feature Engineering

* Created new columns like:

  * `net_amount` (after discount)
  * `year_month`
  * `customer_segment`
  * `is_repeat_customer`

### 3️⃣ Data Analysis

* Total and average sales
* Sales by country and category
* Most sold products (quantity & revenue)
* Payment method usage
* Customer behavior analysis
* 
### 4️⃣ Visualization

* Bar charts (category sales)
* Pie chart (payment methods)
* Line chart (sales trend)
* Histogram (age distribution)
